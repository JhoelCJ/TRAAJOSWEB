⚠️  IMPORTANT - SQL PARA SUPABASE
═══════════════════════════════════════════════════════════════════════════════

Ejecuta en Supabase SQL Editor:

CREATE TABLE employees (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL,
  email TEXT NOT NULL UNIQUE,
  address TEXT,
  cellphone TEXT,
  salary DECIMAL(10, 2) NOT NULL,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

(Luego habilita RLS - ver INICIO_RAPIDO.md)

🐛 SOLUCIÓN RÁPIDA DE PROBLEMAS
═══════════════════════════════════════════════════════════════════════════════

Problema: SupabaseService not found
Solución: Crea Services/SupabaseService.cs (copiar de SupabaseService_CODE.txt)

Problema: Tabla employees no existe
Solución: Ejecuta el SQL en Supabase

Problema: Credenciales inválidas
Solución: Verifica appsettings.json contra Supabase Dashboard

Problema: Puerto ocupado
Solución: Cambia puerto en launchSettings.json
