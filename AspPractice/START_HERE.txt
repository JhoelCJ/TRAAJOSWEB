PASO 2: Configurar Supabase (2 minutos)
───────────────────────────────────────
  Abre: appsettings.json
  Busca: "Supabase": { ... }
  Edita: Con tus credenciales de Supabase
  
  Donde obtener credenciales:
    → https://supabase.com
    → Nuevo proyecto
    → Settings > API
    → Copia URL y keys

PASO 3: Crear tabla en Supabase (1 minuto)
──────────────────────────────────────────
  Abre: Supabase Dashboard
  Ve a: SQL Editor
  Copia: SQL de ALL_FILES_CONTENT.md
  Pega: En editor
  Ejecuta: El SQL

  Luego habilita RLS (ver INICIO_RAPIDO.md)
